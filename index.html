Devil's Playground

A JavaScript idle detection script.

Usage


	var idle = new DevilsPlayground(
		function() {
			console.log('You have been idle for 8 seconds!');
		},
		8
	);
	idle.start();

This code appears on this page. You can open the JavaScript console to see the notification.
